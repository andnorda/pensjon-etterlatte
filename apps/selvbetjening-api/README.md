# selvbetjening-api

API som tilgjengeliggjør data for frontend å kommunisere med diverse apper.

### Request

#### Søknad

`POST /api/soeknad`

#### Kladd

`POST /api/kladd`\
`GET /api/kladd`\
`DELETE /api/kladd`

#### Person

`GET /person/innlogget`

#### Land

`GET /kodeverk/alleland`

#### Internal

`GET /internal/isalive`\
`GET /internal/isready`
